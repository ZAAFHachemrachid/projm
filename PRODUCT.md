# Product

## Register

product

## Users
Developers who manage multiple code repositories and workspaces on their local machine, needing a fast, frictionless way to categorize, group, and navigate between their projects.

## Product Purpose
Automatically organize developer directories into clean category-based folders under a single base directory, while providing lightning-fast command-line and GUI fuzzy-jump navigation.

## Brand Personality
- Efficient: Built for speed and utility.
- Sleek: Dark-themed, modern, and developer-native.
- Solid: Reliable file-system operations with zero data loss.

## Anti-references
- Cluttered SaaS dashboards filled with useless metrics.
- Over-rounded, slow, animated interfaces that feel laggy.
- Cryptic terminal applications without friendly visual fallbacks.

## Design Principles
1. **Speed is King**: Minimize latency in scanning, navigation, and rendering.
2. **Keyboard-First**: Ensure power users can navigate and trigger actions without picking up the mouse.
3. **Harmonious Contrasts**: Maintain high-contrast type and crisp accents without visual pollution.
4. **Context Preservation**: Keep the developer in their flow; never lose current state or context.

## Accessibility & Inclusion
- Full keyboard focusability and clean tab indexing.
- Screen reader friendly labeling on all button elements.
- Clean high-contrast theme supporting dark mode by default.
